# Introduction to ngx-stories

ngx-stories is a powerful Angular component library that brings Instagram-style stories to your Angular applications. Create engaging, interactive story experiences with support for images, videos, and custom Angular components.

## What you can build

With ngx-stories, you can create:

- **Social media feeds** with story carousels
- **Product showcases** with interactive walkthroughs
- **Onboarding flows** using custom components
- **Media galleries** with automatic progression
- **Interactive tutorials** with swipe navigation

## Key features

### Navigation & interaction

- **Swipe gestures**: Navigate between stories and groups with intuitive swipes
- **Tap controls**: Tap left/right to move between stories, hold to pause
- **Keyboard support**: Full keyboard navigation with arrow keys, space, and escape
- **Progress indicators**: Visual progress bars for each story

### Media handling

- **Images**: Automatic preloading and caching for smooth playback
- **Videos**: Auto-play with audio controls and duration detection
- **Components**: Render any Angular component as a story slide
- **Error handling**: Graceful fallbacks when content fails to load

### Customization

- **Gradient backgrounds**: Dynamic, Instagram-like backgrounds based on content colors
- **Configurable dimensions**: Set custom width and height for your story viewer
- **Playback control**: Start at specific stories or groups
- **Event emissions**: Hook into story changes, completion, and user interactions

## Browser compatibility

ngx-stories works in all modern browsers:

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
- Mobile browsers (iOS Safari, Chrome Mobile)

<Note>
ngx-stories requires Angular 18.2.0 or higher. For older Angular versions, check the compatibility table in the installation guide.
</Note>
