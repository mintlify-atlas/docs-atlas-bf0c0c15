# Quick start

Get your first story viewer up and running in minutes. This guide will walk you through creating a basic Instagram-like story experience.

## Build your first story viewer

<Steps>
  <Step title="Import required types and component">
    In your component file, import `NgxStoriesComponent` and the `StoryGroup` interface:

    ```typescript
    import { Component } from '@angular/core';
    import { NgxStoriesComponent, StoryGroup } from 'ngx-stories';
    ```
  </Step>

  <Step title="Create story data">
    Define your story groups with images, videos, or custom components:

    ```typescript
    @Component({
      selector: 'app-root',
      standalone: true,
      imports: [NgxStoriesComponent],
      templateUrl: './app.component.html',
      styleUrl: './app.component.css'
    })
    export class AppComponent {
      readonly storyGroups: StoryGroup[] = [
        {
          name: 'Steve Smith',
          stories: [
            { 
              type: 'image', 
              content: 'https://i.ibb.co/ZMVy3KN/pexels-rpnickson-2486168.jpg' 
            },
            { 
              type: 'video', 
              content: 'https://commondatastorage.googleapis.com/gtv-videos-bucket/sample/BigBuckBunny.mp4' 
            },
            { 
              type: 'image', 
              content: 'https://d38b044pevnwc9.cloudfront.net/cutout-nuxt/enhancer/3.jpg' 
            }
          ]
        },
        {
          name: 'John Doe',
          stories: [
            { 
              type: 'image', 
              content: 'https://d38b044pevnwc9.cloudfront.net/cutout-nuxt/enhancer/4.jpg' 
            },
            { 
              type: 'image', 
              content: 'https://d38b044pevnwc9.cloudfront.net/cutout-nuxt/enhancer/5.jpg' 
            },
            { 
              type: 'video', 
              content: 'https://commondatastorage.googleapis.com/gtv-videos-bucket/sample/BigBuckBunny.mp4' 
            }
          ]
        }
      ];
    }
    ```
  </Step>

  <Step title="Add the component to your template">
    Use the `ngx-stories` component in your HTML template:

    ```html
    <ngx-stories [storyGroups]="storyGroups"></ngx-stories>
    ```
  </Step>

  <Step title="Run your application">
    Start your Angular development server:

    ```bash
    ng serve
    ```

    Open your browser to `http://localhost:4200` and you should see your story viewer!
  </Step>
</Steps>

## Understanding story groups

Stories are organized into groups, where each group represents a collection of stories from a single source (user, brand, etc.).

### StoryGroup interface

```typescript
interface StoryGroup {
  /** Unique identifier. Auto-generated if not provided. */
  id?: string;
  
  /** Display name shown above the story content. */
  name: string;
  
  /** Array of stories in this group. */
  stories: Story[];
}
```

### Story interface

```typescript
interface Story {
  /** Unique identifier. Auto-generated if not provided. */
  id?: string;
  
  /** The type of content this story displays. */
  type: 'image' | 'video' | 'component';
  
  /**
   * The content to display.
   * - For 'image' and 'video' types: URL string
   * - For 'component' type: Angular component class
   */
  content: string | Type<unknown>;
  
  /**
   * CORS setting for image/video elements.
   * Set to null to omit the attribute entirely.
   * @default 'anonymous'
   */
  crossOrigin?: 'anonymous' | 'use-credentials' | '' | null;
}
```

## Adding custom components

You can render any Angular component as a story for maximum flexibility:

```typescript
import { CustomComponentComponent } from './components/custom-component/custom-component.component';

readonly storyGroups: StoryGroup[] = [
  {
    name: 'Steve Smith',
    stories: [
      { type: 'image', content: 'https://example.com/image.jpg' },
      { type: 'component', content: CustomComponentComponent },
      { type: 'video', content: 'https://example.com/video.mp4' }
    ]
  }
];
```

<Info>
Custom components should be self-contained and handle their own styling. They will be rendered within the story viewer's dimensions.
</Info>

## Customizing the viewer

You can customize the story viewer's appearance and behavior using the `options` input:

```typescript
import { NgxStoriesOptions } from 'ngx-stories';

storyOptions: NgxStoriesOptions = {
  width: 400,                      // Width in pixels (default: 338)
  height: 700,                     // Height in pixels (default: 600)
  backlitColor: '#000',            // Background color (default: '#1b1b1b')
  enableGradientBackground: true,  // Dynamic gradient (default: true)
  currentStoryIndex: 0,            // Starting story index (default: 0)
  currentStoryGroupIndex: 0        // Starting group index (default: 0)
};
```

Then pass it to the component:

```html
<ngx-stories 
  [storyGroups]="storyGroups" 
  [options]="storyOptions">
</ngx-stories>
```

## Handling events

Listen to story events to implement custom behavior:

```typescript
onStoryEnd() {
  console.log('All stories completed');
}

onStoryExit() {
  console.log('User exited stories');
}

onStoryChange(eventData: StoryChangeEventData) {
  console.log('Current story:', eventData.currentStory);
  console.log('Story index:', eventData.currentStoryIndex);
}
```

```html
<ngx-stories 
  [storyGroups]="storyGroups"
  (triggerOnEnd)="onStoryEnd()"
  (triggerOnExit)="onStoryExit()"
  (triggerOnStoryChange)="onStoryChange($event)">
</ngx-stories>
```

<Tip>
The `triggerOnStoryChange` event provides detailed information about the current and previous stories, perfect for analytics tracking.
</Tip>

## Navigation controls

Your users can navigate stories using:

### Touch gestures (mobile)
- **Swipe left**: Next story group
- **Swipe right**: Previous story group  
- **Swipe down**: Exit stories
- **Swipe up**: Trigger custom action
- **Tap right side**: Next story
- **Tap left side**: Previous story
- **Hold**: Pause story

### Keyboard (desktop)
- **Arrow right**: Next story
- **Arrow left**: Previous story
- **Space**: Toggle play/pause
- **Escape**: Exit stories
