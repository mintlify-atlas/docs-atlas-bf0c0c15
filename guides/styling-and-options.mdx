## Overview

The ngx-stories component offers extensive customization through the `NgxStoriesOptions` interface. Configure dimensions, colors, starting positions, and visual effects.

## NgxStoriesOptions Interface

Reference: `projects/ngx-stories/src/lib/interfaces/interfaces.ts:77`

```typescript
interface NgxStoriesOptions {
  width?: number;
  height?: number;
  currentStoryIndex?: number;
  currentStoryGroupIndex?: number;
  backlitColor?: string;
  enableGradientBackground?: boolean;
}
```

### Gradient Background

**Property**: `enableGradientBackground`  
**Type**: `boolean`  
**Default**: `true`  
**Description**: Enable dynamic gradient backgrounds based on content colors

```typescript
storyOptions: NgxStoriesOptions = {
  enableGradientBackground: true  // Enable Instagram-like gradients
};
```

**How It Works**:

1. **Color Extraction**: Analyzes image/video to find dominant colors
2. **Gradient Creation**: Creates smooth linear gradient background
3. **Dynamic Updates**: For videos, updates colors during playback

**Implementation**: `projects/ngx-stories/src/lib/ngx-stories.component.ts:831`

```typescript
getDominantColors(source: HTMLImageElement | HTMLVideoElement): string[] {
  // Draw media to canvas (50x50 for performance)
  this.context.drawImage(source, 0, 0, 50, 50);
  const imageData = this.context.getImageData(0, 0, 50, 50);
  
  // Analyze pixels and extract dominant colors
  // Returns top 2 colors for gradient
  return [`rgb(${color1})`, `rgb(${color2})`];
}
```

<Tabs>
  <Tab title="Enabled (Default)">
    ```typescript
    storyOptions: NgxStoriesOptions = {
      enableGradientBackground: true,
      backlitColor: '#1b1b1b'  // Fallback if extraction fails
    };
    ```

    **Result**: Dynamic gradients that match content colors, smooth transitions for videos
  </Tab>
  
  <Tab title="Disabled">
    ```typescript
    storyOptions: NgxStoriesOptions = {
      enableGradientBackground: false,
      backlitColor: '#000000'  // Solid color always
    };
    ```

    **Result**: Solid `backlitColor` for all stories (better performance, less visual flair)
  </Tab>
</Tabs>

<Warning>
Gradient extraction requires CORS-enabled media. Set `crossOrigin: 'anonymous'` on stories from external domains.
</Warning>

## Starting Position Options

### Current Story Index

**Property**: `currentStoryIndex`  
**Type**: `number`  
**Default**: `0`  
**Description**: Initial story index within the starting group

```typescript
storyOptions: NgxStoriesOptions = {
  currentStoryIndex: 2  // Start at 3rd story (zero-indexed)
};
```

**Use Cases**:
- Resume from saved progress
- Deep linking to specific story
- Skip intro stories

**Example: Resume Progress**

```typescript
export class AppComponent {
  storyOptions: NgxStoriesOptions = {
    currentStoryIndex: this.getSavedStoryIndex(),
    currentStoryGroupIndex: this.getSavedGroupIndex()
  };

  getSavedStoryIndex(): number {
    return parseInt(localStorage.getItem('lastStoryIndex') || '0');
  }

  getSavedGroupIndex(): number {
    return parseInt(localStorage.getItem('lastGroupIndex') || '0');
  }

  onStoryChange(data: any) {
    localStorage.setItem('lastStoryIndex', data.currentStoryIndex);
    localStorage.setItem('lastGroupIndex', data.currentStoryGroupIndex);
  }
}
```

### Current Story Group Index

**Property**: `currentStoryGroupIndex`  
**Type**: `number`  
**Default**: `0`  
**Description**: Initial story group index

```typescript
storyOptions: NgxStoriesOptions = {
  currentStoryGroupIndex: 1  // Start at 2nd group
```

**Example: Deep Linking**

```typescript
export class AppComponent implements OnInit {
  storyOptions: NgxStoriesOptions = {};

  ngOnInit() {
    // Parse URL parameters
    const params = new URLSearchParams(window.location.search);
    const groupIndex = parseInt(params.get('group') || '0');
    const storyIndex = parseInt(params.get('story') || '0');

    this.storyOptions = {
      currentStoryGroupIndex: groupIndex,
      currentStoryIndex: storyIndex
    };
  }
}
```

**URL**: `https://example.com/stories?group=2&story=1`  
**Result**: Opens 3rd group, 2nd story

## Performance Considerations

### Gradient Background Performance

<Info>
**Image Stories**: Colors extracted once when story loads  
**Video Stories**: Colors updated every 30 frames (~2 times per second) for smooth transitions
</Info>

Reference: `projects/ngx-stories/src/lib/ngx-stories.component.ts:781`

```typescript
startVideoBackgroundUpdate(videoElement: HTMLVideoElement) {
  let frameCount = 0;
  const update = () => {
    // Update colors every 30 frames (throttled for performance)
    if (frameCount % 30 === 0) {
      this.updateBackground(videoElement);
    }
    
    // Smooth interpolation every frame
    this.currentColors = [
      this.lerpColor(this.currentColors[0], this.targetColors[0], 0.03),
      this.lerpColor(this.currentColors[1], this.targetColors[1], 0.03)
    ];
    
    frameCount++;
    this.videoFrameId = requestAnimationFrame(update);
  };
}
```

**Optimizations**:
- Canvas size: 50×50 (not full resolution)
- `willReadFrequently: true` context flag
- Runs outside Angular zone (no change detection)
- Direct DOM manipulation for gradient updates

### Disable for Better Performance

```typescript
storyOptions: NgxStoriesOptions = {
  enableGradientBackground: false  // Better performance
};
```

<Tip>
If targeting low-end devices or experiencing performance issues, disable gradient backgrounds.
</Tip>

## Responsive Sizing

### Dynamic Dimensions

```typescript
export class AppComponent {
  storyOptions: NgxStoriesOptions;

  constructor() {
    this.storyOptions = {
      width: this.getResponsiveWidth(),
      height: this.getResponsiveHeight()
    };
  }

  getResponsiveWidth(): number {
    const screenWidth = window.innerWidth;
    if (screenWidth < 768) {
      return Math.min(screenWidth - 40, 338);  // Mobile
    } else if (screenWidth < 1024) {
      return 400;  // Tablet
    } else {
      return 500;  // Desktop
    }
  }

  getResponsiveHeight(): number {
    const screenHeight = window.innerHeight;
    return Math.min(screenHeight - 100, 900);
  }
}
```
